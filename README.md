**Online Voting and Tabulation System :-**

**DESCRIPTION -**	Develops a digital voting platform using Vb.Net with MS Access database to provide a secure, efficient, and transparent voting process. This system is designed to replace the traditional manual voting and vote counting methods, which are time-consuming and prone to errors. The application allows registered users to log in, view candidates based on their district, cast their vote securely, and store voting data in a centralized database. The admin module enables real-time vote tabulation and result management, ensuring accuracy and faster decision-making. This system aims to improve reliability, reduce human effort, and enhance the overall election management process.

**KEY FEATURES -**

•	Provides a digital voting platform using VB.NET.

•	Secure user login using username and password.

•	Displays candidates based on the voter’s district.

•	Allows voters to cast only one vote.

•	Stores voter and voting details securely in a database.

•	Automatically counts votes without manual intervention.

•	Generates accurate tabulation reports.

•	Reduces time required for result declaration.

•	Improves data security and system reliability.

•	User-friendly interface for both voters and admin.

**TECHNOLOGIES USED -**

**Front End :** Visual Basic Professional 

**Back End :** MS Access 2021
